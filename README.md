Menu Nutrition Calculator
Summary of the Project
The menu nutrition calculator is an easy python work designed to tell users to track the nutrition intake of their meal on the preset weekly menu. It stores a detailed, day-wise, and meal-wise menu with related calorie, protein, carbohydrates, and fat content of each item. The script remind the user to the day and meal they are interested in, display the available items, asks for the quantity  for each item, and then evaluates and presents the total nutritional value of their preffered meal.
Features
Weekly menu data. stores all data for meals for all seven days i.e  meal-specific tracking. Supports tracing for calories, protein, carbohydrates, and fat. Nutritional details. Evaluates the total intake of calories, protein, carbohydrates, and fat. Interactive inserting. advise the user through selecting the day, meal, and entering the quantity  for each item.
Technologies/Tools Used
Language-Python 3x Data Structure-Python dictionaries are used widely to store the  structure of the menu.
Steps to install and run the project
Since this is a single script with no outer colony, installation is straight. Save the python code into a file named nutritioncalculator.py
Run the script open on your terminal, navigate to the directory where you saved the file, and execute the script using the python interpreter.
Instruction for testing-run the program-Enter any day, enter meal check in displayed menu, enter quantity. Program will display total nutrition
