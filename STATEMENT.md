Problem Statement
The main problem displayed by this project is the manual, mistakes and time wasting process of evaluating the total intake of nutrition from a many-item meal, based on a fixed, planned weekly menu. Without a tool, users would need to manually look up or sum values of each item for a preferred day and meal, viewing the quantity used for each.
Scope of the Project
The goal of this project is  to provide a  friendly, cooperative utility for evaluating and reporting the average nutritional values of a single meal any given day of the week.
The project consists of storing static, hard menus and its related nutrional data .Handling user input for the day of the week and the meal . Displaying the available  items for the preferred day and meal. Collecting input from the user for each item. Performing the sum of nutritional values on quantity. Outputting the final evaluated totals for the selected meal. The project does not include a graphical user interface, menu update, user login data,  or ideas for daily nutritional goals.
Target users
The primary users for this simple nutrition calculator  are-individuals managing a fixed daily menu who want a way to estimate their nutrition intake. Diet holders or fitness trainers who may use this as a basic or help for planned  indian meals. Individuals who are going through any health related problem and want to restrict or consume extra nutrition. Makers or students learning Python who need a example of working with, user input, basic data evaluation, and arithmetic evaluations.
High-level menu storage-Maintain a well structured, week-long menu covering all meal  with detailed nutrition contents. Meal selection-Allow users to prioritize the day and meal  they wish to evaluate nutrition for.Quantity input-remind the user to input the quantity utilised for every item available in the selected meal. Total nutritional evaluation-Compute the average of calories, protein, carbohydrates, and fat for the entire meal based on the entered quantities. Input Validation-check if the inputted day and meal are present in the menu data.
